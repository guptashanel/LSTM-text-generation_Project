Sports Text Generation using LSTM
Description

This project builds a text generation model using LSTM (Long Short-Term Memory) in TensorFlow/Keras.
The model is trained on sports-related text and generates new text based on a given seed sentence.

Technologies Used

Python

TensorFlow / Keras

NumPy

How to Run

Open the notebook in Jupyter Notebook or VS Code
install dependencies:

pip install tensorflow numpy

Run all cells in order

Example

Input:
sports have

Output:
sports have always been a powerful part of human life ...
